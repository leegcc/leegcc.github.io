公示信息 #

所有人（包括未登录用户）都可查看公示信息。

公示信息主要包含：

  • 热门爱好
  • 爱好广泛用户排行
  • 本月内总获得积分排行
  • 活跃用户列表

热门爱好 #

用户数多的爱好排在前面，少于 2 个用户数的爱好不显示。

爱好用户数
游泳20
摄影18
......

爱好广泛用户排行 #

爱好数多的用户排在前面，少于 2 个爱好的用户不显示。

包含列：

  • 用户姓名：没有设置姓名的用户显示 “无名氏”
  • 爱好数：用户的爱好数
用户姓名爱好数
张三20
李四12
无名氏10
......

本月内总获得积分排行 #

本自然月内总获得积分数多的用户排在前面，只显示排行前 3 的用户。

包含列：

  • 用户姓名
  • 总获得积分：本自然月内获得总积分数
用户姓名总获得积分
张三120
李四80
王五60

活跃用户列表 #

只显示活跃用户，活跃用户的定义为：最近 3 日有参与抽奖的用户。

包含列：

  • 用户姓名
  • 抽奖次数：最近 3 日有参与抽奖的次数
  • 获得积分数：最近 3 日参与抽奖总获得积分
  • 消耗积分数：最近 3 日通过积分兑礼所消耗的积分
